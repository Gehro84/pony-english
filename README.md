# 🐴 Pony English

Eine spielerische Englisch-Lern-App für die 6. Klasse (Schweiz, Lehrmittel «Young World 4»).

- Vokabel-Quiz mit XP und Leveln
- Tages-Challenge mit doppelten XP und 🔥-Serie
- Pferde als Belohnung freischalten
- Läuft auf Handy, Tablet und PC im Browser – Fortschritt wird lokal gespeichert

Neue Vokabeln kannst du in `words.js` ergänzen, die Pferde stehen in `pferde.js`.
